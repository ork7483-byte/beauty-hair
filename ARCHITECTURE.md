## 2. 대시보드 섹션별 상세 구조 & 연결 관계

### S1. 배너 캐러셀 (6슬라이드, 자동 5초 전환)

| 슬라이드 | 내용 | 연결 페이지 |
|---------|------|-----------|
| 1 | 카드 등록하면 1개월 무료 체험 시작! | → `/pricing` |
| 2 | 원장님! AI 헤어 모델컷이 처음이신가요? | → `/app/guide` |
| 3 | 대충 찍어도 OK! AI 헤어 모델컷 만들기 | → `/app/hair-model` |
| 4 | 모든 SNS에 헤어 사진 한 번에 올리기 | → `/app/connect-naver` |
| 5 | 네이버 플레이스 쓰시는 원장님 주목! | → `/app/naver-place` |
| 6 | 매주 SNS에 올릴 모델컷과 글 받아보기 | → `/app/auto-posting` |

**구조**: 각 슬라이드가 핵심 기능/유틸리티 페이지의 **진입점** 역할. 사용자를 깊이 있는 페이지로 유도.

---

### S2. 내가 만든 사진 (가로 스크롤 갤러리)

```
[대시보드 S2] ──── 전체 보기 ────→ /app/gallery
                   썸네일 클릭 ────→ /app/gallery (개별 이미지)
```

- **역할**: 사용자가 생성한 결과물의 **즉각적 피드백** 제공
- **원본**: 사이드바 상단에도 동일 썸네일 존재 (데스크톱)
- **데이터**: 최근 생성된 이미지 최대 8장

---

### S3. 내 SNS 사진 관리 (연동 플랫폼 카드)

```
[대시보드 S3] ──── 전체 보기 ────→ /app/connect-naver
                   카드 클릭 ─────→ /app/connect-naver
                   CTA 클릭 ─────→ /app/connect-naver (연동 플로우)
```

- **역할**: 외부 플랫폼 연동 상태 **대시보드 위젯**
- **3열 스탯**: 방문자 추이 / 리뷰 TOP3 / 사진 등록 필요
- **CTA**: "네이버 플레이스 연동하고 사진 바로 올리기"
- **연동 전/후**: 연동 전이면 CTA 표시, 연동 후면 실제 데이터 표시

---

### S4. 100% 무료로 시작해 보세요 (핵심 기능 2개 카드)

```
[대시보드 S4]
  ├── 카드1: AI 헤어 모델컷 생성 ──→ /app/hair-model
  │   배지: "헤어 AI 사진 1위"
  │   포인트: AI 모델 초상권 무료, SNS 사이즈 자동, 10초 생성
  │
  └── 카드2: 스타일 시뮬레이션 ────→ /app/style-simulation
      배지: "한 번에 최대 5스타일"
      포인트: 미리보기, 5가지 비교, 상담용 이미지
```

- **역할**: 핵심 가치 제안 + **전환(conversion) 유도**
- **레이아웃**: 모바일 세로 스택 / 데스크톱 가로 2열
- **연결**: 각 카드 → 해당 기능의 상세 페이지 (에디터/업로드 화면)

---

### S5. 프로모션 배너 (CTA)

```
[대시보드 S5] ──── 클릭 ─────────→ /pricing
```

- **역할**: 유료 전환 **CTA 배너**
- **내용**: "프로 요금제 1개월 무료 체험 시작! 월 19,800원 → 0원"
- **위치**: S4(무료 기능 소개) 바로 아래 → 자연스러운 업셀 흐름

---

### S6. SNS 마케팅 사진 만들기 (3개 항목 리스트)

```
[대시보드 S6]
  ├── 인스타 피드 ─────────→ /app/sns-feed
  │   "1:1 정사각형 피드 최적화 사진"
  │
  ├── 릴스 / 숏폼 ─────────→ /app/reels
  │   "9:16 세로형 숏폼 콘텐츠"
  │
  ├── 네이버 플레이스 ──────→ /app/naver-place
  │   "네이버, 카카오맵 등 지도앱 사진"
  │
  └── 연동 안내 배너 ───────→ /app/connect-instagram
      "인스타그램 연동하시는 원장님 주목!"
```

- **역할**: SNS 플랫폼별 **맞춤 사이즈 생성** 기능 안내
- **레이아웃**: 모바일 세로 리스트(h-90px, 이미지+텍스트) / 데스크톱 3열 카드
- **연결**: 각 항목 → 해당 SNS 사이즈에 맞춘 에디터 페이지

---

### S7. 더 다양한 AI 기능 모아보기 (2x4 그리드)

```
[대시보드 S7]
  ├── SNS 콘텐츠 ──────→ /app/sns-content
  ├── 포스터 ──────────→ /app/poster
  ├── 움직이는 사진 ───→ /app/animated
  ├── 헤어 컬러 변환 ──→ /app/color-change    [NEW]
  ├── 해상도 높이기 ───→ /app/upscale
  ├── 조명 & 그림자 ───→ /app/lighting
  ├── 텍스트 추가 ─────→ /app/text-overlay
  └── SNS 정기 포스팅 ─→ /app/auto-posting    [자동 업로드 설정]
```

- **역할**: 부가 기능 **디스커버리**
- **레이아웃**: 모바일 2열 / 데스크톱 4열
- **카드 구조**: 이미지(h-113px) + 제목(p-4) + 선택적 배지

## 5. 사이드바 ↔ 대시보드 섹션 매핑

```
사이드바 메뉴              대시보드 섹션에서의 노출 위치
─────────────────────────────────────────────────────
[내가 만든 사진]           → S2 (가로 스크롤 갤러리)

추천 기능
  AI 모델컷 생성           → S4 카드1 (100% 무료로 시작)
  스타일 시뮬레이션        → S4 카드2 (100% 무료로 시작)
  SNS                      → S7 그리드 (더 다양한 AI 기능)
  포스터                    → S7 그리드
  움직이는 사진             → S7 그리드
  배경 제거                 → (S4 대체 가능)

SNS 마케팅 사진
  인스타 피드               → S6 리스트 (SNS 마케팅 사진)
  릴스 / 숏폼              → S6 리스트
  네이버 플레이스           → S6 리스트

AI가 관리하는 내 매장
  SNS 정기 포스팅           → S7 그리드 (자동 업로드 설정 배지)
  AI 트렌드 리포트          → (대시보드 직접 노출 없음)

연동
  네이버 플레이스 연동      → S3 카드 (내 SNS 사진 관리)
  인스타그램 연동           → S6 하단 배너 (연동 안내)

더 다양한 AI 기능
  헤어 컬러 변환            → S7 그리드 (NEW 배지)
  해상도 높이기             → S7 그리드
  조명 & 그림자             → S7 그리드
  텍스트 추가               → S7 그리드
```
